Development Process

Designing mobile first is key to a successful responsive design. Once built out in code the browser window can gradually be expanded. Working this way reduces back-and-forth revisions between devices and keep code in media queries minimal.

To help manage my files and allow for version control I created a GitHub repository. Since the demo contains compressed images and minified code, you'll probably want to have a look at the original developer's files. Have a look at the repository or download the project package:
